Comparacion de: Modelo OSI y TCP

OSI
Concepto:
OSI proporciona un estándar para que distintos sistemas de equipos puedan comunicarse entre sí.
Características:
  • Físico: Define las características físicas de la red material.
  • Gestiona las conexiones de red para las capas superiores.
  • Enlace: Proporciona el servicio de envió de datos a través del enlace físico.
  • Sesión: Gestiona las concesiones entre aplicaciones cooperativas.
  • Transporte: Proporciona servicios de detección y corrección de errores.
  • Aplicación: Aplicaciones que usan la red
  • Presentación: Estandariza la forma en la que se presentan los datos a las aplicaciones
Origen:
El modelo OSI se creó para solucionar el problema de que las redes incompatibles y no se comunicaban entre sí debido a distintos softwares y hardwares
