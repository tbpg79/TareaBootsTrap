¿Que es el Modelo OSI

El modelo OSI (Open Systems Interconnection) es un modelo de referencia para protocolos de comunicación de redes de computadoras.

Este modelo se divide en siete capas, cada una de las cuales representa un conjunto de funciones de comunicación específicas. Las capas del modelo OSI son:

  • Física
  • Enlace de datos
  • Red
  • Transporte
  • Sesión
  • Presentación
  • Aplicación

Concepto:

El modelo OSI proporciona un estándar para que distintos sistemas de equipos puedan comunicarse entre sí.

Características:


Físico: Define las características físicas de la red material.


Red: Gestiona las conexiones de red para las capas superiores.


Enlace: Proporciona el servicio de envío de datos a través del enlace físico.


Sesión: Gestiona las concesiones entre aplicaciones cooperativas.


Transporte: Proporciona servicios de detección y corrección de errores.


Aplicación: Aplicaciones que usan la red.


Presentación: Estandariza la forma en la que se presentan los datos a las aplicaciones.

¿Que es el Protocolo TCP/ip

Protocolo de enlace de datos que se usa en Internet para que los ordenadores y otros dispositivos envíen y reciban datos.

Concepto:

Protocolo de enlace de datos que se usa en Internet para que los ordenadores y otros dispositivos envíen y reciban datos.

Características:

  • Es un protocolo que funciona mediante la conexión mutua entre cliente y servidor.
  • Ordena los segmentos provenientes del protocolo IP.
  • Monitorea el flujo de los datos y permite evitar la saturación de la red.
  • Entrega los datos al protocolo IP en forma de segmentos de longitud variable.
  • Permite circular de forma simultánea a la información proveniente de diferentes fuentes.

Origen:

Surgió de un proyecto de defensa llamado DARPA en 1969. En 1983 el nuevo conjunto de protocolos TCP/IP fue adoptado como estándar y finalmente se convirtió en el más usado en redes y el protocolo estándar de internet.
